# ESP32 MQTT Realtime Monitoring Dashboard

Project IoT menggunakan:

- ESP32
- MQTT Mosquitto
- Node-RED
- MySQL / MariaDB
- OLED SSD1306
- BMP280
- DHT11

## Features

- Realtime Monitoring
- Data Logging
- Node-RED Dashboard
- MQTT Communication
- MySQL Storage
- Responsive Dashboard

## Hardware

- ESP32
- DHT11
- BMP280
- OLED SSD1306

## Dashboard Preview

Node-RED dashboard menampilkan:
- Temperature
- Humidity
- Pressure
- Realtime Chart

## Import Flow

Import file:

```text
flows.json
```

ke Node-RED.

## MQTT Topic

```text
sensor/data
```
